
If you believe you have found a security vulnerability in this repository which can be potentially harmful for the users in anyway, please do not report security vulnerabilities through public GitHub issues. Instead, please report it to us as described below.

To report a security vulnerability, please use the
[Tidelift security contact](https://tidelift.com/security).
Tidelift will coordinate the fix and disclosure.

or raise it on Github under security advisories section, do not raise it as public issue.
